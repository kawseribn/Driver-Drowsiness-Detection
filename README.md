# Driver Drowsiness Detection (EYE(blinking, gaze), yawning)


## Installation

Clone this project:

```
git clone https://github.com/kawseribn/Driver-Drowsiness-Detection.git
```

Install these dependencies (NumPy, OpenCV, Dlib):

```
pip install -r requirements.txt
```

Run the demo:

```
python example.py
```
